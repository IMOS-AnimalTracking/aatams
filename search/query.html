Data Source (Web Feature Service):
  1. Select a dataset of interest from the Datasets list.
  2. Select or enter Filter Parameters for the respective dataset selected.
  3. Choose the number of records from the dataset to retrieve from the server (max. limit = 10000).
  4. Select a display format for the results to be displayed in within the bottom frame of this window.
  5. Click a button below to either: display all records (up to the max. limit), or, display a total record count (with no limit).
